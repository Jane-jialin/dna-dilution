# DNA 稀释计算器

一个简洁实用的 DNA 样品稀释加水量计算工具，支持电脑和手机浏览器使用。

---

## 功能简介

根据稀释公式 **C₁ × V₁ = C₂ × V₂**，输入当前 DNA 浓度，自动计算将样品稀释至目标浓度所需添加的纯水体积。

| 字段 | 说明 | 默认值 |
|---|---|---|
| 当前 DNA 浓度 | **用户填写**，支持小数 | — |
| 样品体积 | 可修改 | 10 μL |
| 目标浓度 | 可修改 | 1 ng/μL |
| **需添加纯水量** | **自动计算结果** | — |

---

## 界面特性

- **一屏操作**：浓度输入与结果展示位于同一卡片内，无需滚动
- **实时计算**：输入浓度后即时显示结果，无需点击按钮
- **一键清除**：输入框右侧提供清除按钮，快速重置
- **参数可调**：样品体积和目标浓度折叠在设置面板中，点击展开修改
- **结果汇总**：显示样品体积、加水量、终体积三项数据
- **公式展示**：底部显示完整的公式代入过程，方便核验
- **错误提示**：浓度低于目标值或输入无效时给出明确提示
- **中英双语**：自动检测系统语言，中文环境显示中文，其他环境显示英文
